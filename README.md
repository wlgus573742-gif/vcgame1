# 심해 진주 줍기 - 수족관 물소리 BGM 개선 버전

## 포함 파일
- index.html
- style.css
- script.js
- README.md

## 이번 버전 변경점
- 방향키 상하좌우 조작 방식은 유지했습니다.
- 효과음은 기존처럼 유지했습니다.
- BGM을 더 맑고 청아한 수족관 물소리 느낌으로 개선했습니다.
- 낮게 깔리는 어두운 드론음을 줄이고, 물방울 / 찰랑거림 / 수조 순환 물소리 중심으로 변경했습니다.
- 유튜브 음원을 직접 사용하지 않고, script.js에서 Web Audio API로 비슷한 질감의 사운드를 직접 생성합니다.

## 적용 방법
기존 vcgame1 폴더의 index.html, style.css, script.js를 이 파일들로 교체한 뒤, VS Code Source Control에서 커밋 메시지를 아래처럼 적고 커밋/동기화하세요.

커밋 메시지:
수족관 물소리 BGM 개선

또는 터미널을 사용한다면 아래 명령어를 입력하세요.

```bash
git add .
git commit -m "수족관 물소리 BGM 개선"
git push
```
