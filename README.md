# 심해 진주 줍기 - 수족관 물소리 BGM 개선 버전

## 포함 파일
- index.html
- style.css
- script.js
- README.md

## 이번 버전 변경점
- 방향키 상하좌우 조작 방식은 유지했습니다.
- 효과음은 기존처럼 유지했습니다.
- BGM을 더 맑고 청아한 수족관 물소리 느낌으로 개선했습니다.
- 낮게 깔리는 어두운 드론음을 줄이고, 물방울 / 찰랑거림 / 수조 순환 물소리 중심으로 변경했습니다.
- 유튜브 음원을 직접 사용하지 않고, script.js에서 Web Audio API로 비슷한 질감의 사운드를 직접 생성합니다.

## 추가 업데이트
- 승리 엔딩 사운드를 더 밝은 물방울 팡파르와 반짝이는 상승 멜로디로 강화했습니다.
- 패배 엔딩 사운드를 물속에서 가라앉는 듯한 하강음과 둔탁한 충돌음으로 강화했습니다.

## 추가 업데이트
- 게임 엔딩에 닉네임 입력 기능을 추가했습니다.
- 승리/패배 후 닉네임을 입력하면 플레이 로그가 랭킹 TOP 10에 저장됩니다.
- 랭킹에는 닉네임, 성공/실패 여부, 점수, 남은 생명, 플레이 시간이 표시됩니다.
- 랭킹 데이터는 브라우저 localStorage에 저장됩니다.
  - 같은 PC/브라우저에서는 기록이 유지됩니다.
  - 다른 사람의 기기와 실시간 공유되는 온라인 랭킹은 아닙니다.
  - 온라인 공유 랭킹은 Firebase/Supabase 같은 데이터베이스 연결이 필요합니다.

## 추천 커밋 메시지
랭킹 시스템 추가


## 추가 업데이트
- 게임 난이도를 1단계, 2단계, 3단계로 나누었습니다.
- 점수 0~4점: 1단계
  - 장애물 2개
  - 기본 속도
  - 비교적 여유 있는 충돌 판정
- 점수 5~9점: 2단계
  - 장애물 3개
  - 장애물 이동 속도 증가
  - 충돌 판정 난이도 상승
- 점수 10~15점: 3단계
  - 최고 난이도
  - 장애물 이동 속도 추가 증가
  - 다이버 이동 속도 소폭 감소
  - 진주 배치 안전거리 감소
- 랭킹 기록에 도달 단계가 함께 표시됩니다.
