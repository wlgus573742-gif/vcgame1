# 심해 진주 줍기 - 효과음/BGM 버전

## 포함 파일
- index.html
- style.css
- script.js

## 적용된 사운드
- 게임 시작: 물속 진입 느낌의 상승음
- 점프/이동: 짧은 물방울 효과음
- 진주 획득: 반짝이는 차임 효과음
- 장애물 충돌: 물속 둔탁한 충돌음
- 승리: 밝은 성공 멜로디
- 게임 오버: 낮아지는 실패음
- BGM: 심해 드론 + 랜덤 물방울 앰비언스

## 사용 방법
기존 GitHub 저장소의 index.html, style.css, script.js를 이 파일들로 교체한 뒤 아래 명령어를 실행하세요.

```bash
git add .
git commit -m "효과음과 BGM 추가"
git push
```
